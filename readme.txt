== Description ==

Custom Login Redirect adds a drop down to the login form which lets you to redirect to custom locations after logging in. There are 16 default options already present in the drop down and you can add your own custom URls as well.

Visit <a href="http://www.priteshgupta.com/plugins/custom-login-redirect" target="_blank">Custom Login Redirect Plugin Homepage</a> for more information.

By default, the drop down options include:

* Dashboard
* Last Page
* Front Page
* Add New Post
* Manage Posts
* Add New Page
* Manage Pages
* Upload New Media
* Media Library
* Edit Comments
* Manage Themes
* Manage Widgets
* Edit Themes
* Manage Plugins
* Install Plugins
* General Settings

If you don't want any of these option(s) you can simply disable them from the Plugin Settings Menu in the Dashboard.

== Installation ==

1. Download the latest version
2. Extract it in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Customize the plugin in the Settings > Custom Login Redirect Plugin Options

Visit <a href="http://www.priteshgupta.com/plugins/custom-login-redirect" target="_blank">Custom Login Redirect Plugin Homepage</a> for more information.
